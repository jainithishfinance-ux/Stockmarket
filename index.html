Notes for —


Trend Model
Regression-based trend predictor trained on 6 months of OHLCV data using 14 engineered features including RSI, MACD, Bollinger Band position, ATR, SMA distances, and momentum returns. Predicts next-day price direction and magnitude.
Classification Model
Binary UP/DOWN classifier using logistic regression on the same feature set. Outputs direction probability with confidence score. Trained via gradient descent with 300 epochs on bootstrapped samples.
AI Analysis Layer
Large language model analysis combines live price data, technical indicators, and ML model outputs to produce multi-horizon forecasts (1D/1W/1M), support/resistance levels, risk assessment, and plain-language reasoning.
Data Sources
📊 Yahoo Finance — OHLCV data, real-time quotes, fundamentals. Accessed via Python backend.
📰 Yahoo Finance News — Company-specific news feed, updated in real time.
🔁 Live Refresh — Quote data auto-refreshes every 60 seconds while viewing a stock.
